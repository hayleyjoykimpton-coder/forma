FORMA v1

FILES
- app/: installable Progressive Web App
- design/: design-system HTML and tokens JSON

RUN LOCALLY
Open app/index.html in a browser for a quick preview.

INSTALL ON IPHONE
1. Host the contents of app/ on an HTTPS host such as Netlify Drop or GitHub Pages.
2. Open the hosted URL in Safari.
3. Share -> Add to Home Screen.

DATA
This build saves data locally in the browser on each device. It does not yet sync between devices.

NOT INCLUDED YET
- Cloud account and sync
- Push notifications
- Real AI meal analysis
- Automatic OCR from InBody photos
